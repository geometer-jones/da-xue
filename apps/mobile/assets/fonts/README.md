Bundled Chinese fonts for the mobile app:

- `NotoSansCJKsc-Regular.otf`
  - Source: `notofonts/noto-cjk`
  - Used for the packaged sans-style options
- `NotoSerifCJKsc-Regular.otf`
  - Source: `notofonts/noto-cjk`
  - Used for the packaged serif-style options
- `LXGWWenKai-Regular.ttf`
  - Source: `lxgw/LxgwWenKai`
  - Used for the packaged Kai-style option

These files are bundled so Chinese typography remains available even when the
device does not ship the corresponding system fonts.
